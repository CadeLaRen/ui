= Ninja ui

Ninja user interface is skilled in the techniques of JavaScript and CSS, going unnoticed until called upon to perform the arts of webjutsu.

1. gem install ninjaui
2. cd PATH/TO/YOUR/WEBAPP
3. ninjaui

Visit http://ninjaui.com[http://ninjaui.com/] for how to include, use, and customize.
